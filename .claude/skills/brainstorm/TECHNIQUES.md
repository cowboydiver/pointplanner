# Idea-generation techniques

Reach for one of these when the well runs dry, ideas converge too early, or the user wants a jolt in a new direction. Name the technique out loud, run it for a round, then return to free-form riffing. Mix them — don't grind one into the ground.

## Reframers — change the question

- **How Might We** — restate any stuck point as "How might we…?" Broaden ("…delight new users?") or narrow ("…cut signup to one tap?") to find the productive altitude.
- **Five Whys** — ask "why" five times to reach the real problem under the stated one, then brainstorm the root instead of the symptom.
- **Flip it / Reverse brainstorm** — ask "how could we make this *worse*?" or "how do we guarantee failure?" Then invert each answer into a fix.
- **Worst possible idea** — deliberately generate terrible ideas. They're freeing, funny, and often invertible into something great.

## Expanders — generate volume fast

- **Crazy 8s** — eight ideas in eight minutes (or eight rapid lines). Speed kills the inner critic.
- **SCAMPER** — run the subject through seven lenses: **S**ubstitute, **C**ombine, **A**dapt, **M**odify/Magnify, **P**ut to other use, **E**liminate, **R**everse. One or two ideas per lens.
- **Mash-up** — force-combine two ideas already on the board ("what's the lovechild of 3 and 9?"), or combine your topic with a random noun.
- **More like this / less like this** — pick a promising idea and spawn ten variations along a dial (bigger, smaller, weirder, cheaper, slower).

## Provokers — break the pattern

- **Random stimulus** — grab a random word, object, or image and force a connection to the challenge. Generates surprising bridges.
- **Analogies / "How would X do it?"** — how would Disney, a pirate crew, a kindergarten teacher, or nature solve this? Borrow the mechanics.
- **Constraints** — impose an artificial limit: "if it had to cost $0", "if it shipped tomorrow", "if it used no screens". Constraints breed creativity.
- **Extremes** — solve for the most extreme user (a toddler, a billionaire, someone with 3 seconds). Edge cases reveal new directions.

## Convergers — sort the pile

- **Affinity clustering** — group raw ideas into emergent themes; the clusters themselves suggest new ideas in the gaps.
- **Dot voting** — user spends a fixed number of "stars" across the ideas to surface favorites without debate.
- **Effort/impact** — plot the finalists on easy↔hard vs low↔high impact; the top-left quadrant (easy + high impact) wins first.
- **NUF test** — score each finalist on whether it's **N**ew, **U**seful, and **F**easible.
